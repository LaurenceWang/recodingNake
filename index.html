Recoding Frieder Nake's Walk-Trough-Raster Vancouver

This is a recoding project for our Algorithmic Aesthetics class at IMAC made by Laurence Wang and Elia Roard.
This project is based on Frieder Nake's work you can find here

This piece is composed of 3 layers of grids in 3 different colors :

  • yellow (first)
  • orange (second)
  • blue (third)

The grids are 40 x 40 and are made of squares of 10 x 10. They destructure themselves as they approach the right and bottom side. The orange grid als forms a heap near the bottom left edge and isn't aligned with the rest.

Frider Nake wrote algorithm that dictated certain parameters of an image but left other variables to be determined at random. That is why for our recoding project, we decided to do the same.
We used a Gaussian function and a random number drawned between 0 and 1 to help position the squares of the different layers.

To extend this project, we have made interactive versions of our code to help you understand the process behind this project better.
