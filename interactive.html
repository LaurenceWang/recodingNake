Interactive layers

back

The algorithm is built using three different layers, that's what we saw first when we had to describe this artwork. This section illustrates this fact by letting the users rearrange the composition with different means of interaction.

By clicking on the output, it puts the layer closer to the user in the back of the artwork. It shows the importance of order between the layers

By hovering on the output the layers separate themselves. It reveals that one line can hide another, it also shows grid alignment and how layers interact differently when their grid is not aligned.

Square border thickness is a factor that changes the harmony of the artwork. In this section, the user can gradually change the border thickness to realise how it modifies the output.
